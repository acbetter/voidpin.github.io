无序列表

HTML Tag: <ul>

Note: 我们只使用 -. 不使用 + 和 *。

* 和列表项直接保留一个空格，有序列表同理。

  • Red

- Green

有序列表

HTML Tag: <ol>

  1. Bird
  2. McHale
  3. Parish
1. Bird
2. McHale
3. Parish

特定写法会导致渲染错误，所以我们需要加入 backslash-escape (\) 反斜杠。

  1. What a great season.

1986. What a great season.

1986. What a great season.

1986\. What a great season.

行内 span

行内元素的主要特征是不换行。

For any markup that is not covered by Markdown’s syntax, you simply use HTML itself. There’s no need to preface it or delimit it to indicate that you’re switching from Markdown to HTML; you just use the tags.

Note that Markdown formatting syntax is not processed within block-level HTML tags.

Unlike block-level HTML tags, Markdown syntax is processed within span-level tags.

Work

**Not Work**
**Not Work** 
<span>**Work**</span>

<div>**Not Work**</div>

<div>
  **Not Work**
</div>

颜色

颜色自带加粗，使用时既不需要使用 <b></b> 标签，也不需要使用 **x**。

  • Junk <span class="gw2b-junk">Junk</span>
  • Basic <span class="gw2b-basic">Basic</span>
  • Fine <span class="gw2b-fine">Fine</span>
  • Masterwork <span class="gw2b-masterwork">Masterwork</span>
  • Rare <span class="gw2b-rare">Rare</span>
  • Exotic <span class="gw2b-exotic">Exotic</span>
  • Ascended <span class="gw2b-ascended">Ascended</span>
  • Legendary <span class="gw2b-legendary">Legendary</span>

除了上述的品质颜色，我们还添加了下面几种常用颜色。

  • Dark <span class="gw2c-dark">Dark</span>
  • Green <span class="gw2c-green">Green</span>
  • Yellow <span class="gw2c-yellow">Yellow</span>
  • Orange <span class="gw2c-orange">Orange</span>
  • Red <span class="gw2c-red">Red</span>
  • Redd <span class="gw2c-redd">Redd</span>
  • Blue <span class="gw2c-blue">Blue</span>
  • Gray <span class="gw2c-gray">Gray</span>
  • Blue Gray <span class="gw2c-blue-gray">Blue Gray</span>
