投稿须知

本站以 Markdown 为主要书写语言。

建议使用 Visual Studio Code 作为文本编辑器。在安装好 VSCode 后，你 必须 安装如下插件：

  • Markdown All in One
  • markdownlint

其他可选插件有：

  • Auto Close Tag
  • Auto Rename Tag
  • Auto Complete Tag
  • Even Better TOML

在书写 Markdown 文件中，应尽量遵循以下最佳实践：

  1. 不滥用 Html 代码，防止后期维护困难
  2. 不滥用 表格，https://www.lifewire.com/dont-use-tables-for-layout-3468941
  3. 尽量使用 armory-embeds 嵌入激战相关物品/技能/特性等
  4. 嵌入激战相关物品/技能/特性等时，建议在 GW2 Wiki 和 GW2 Item Search 查找 ID，并通过 Form:Search by id 验证其 ID 可用性。
  5. 可以通过 该链接 获得 Item Stat 装备属性对应 ID，比如 Berserker 的 ID 为 584。
  6. 可以通过 该链接 获得 Specialization 职业特长对应 ID，比如 Spellbreaker 的 ID 为 61
  7. 可以通过 该链接 获得 Traits 技能特性对应 ID，比如 的 ID 为 637
  8. 使用 API ID copy tool on wiki.guildwars2.com 可以高效复制 ID
