迷雾碎层：配装

本文介绍了来自 Discretize(DT) 网站的所有职业配装 （Builds），帮助你根据你的游戏风格选择适合自己的职业。

  • 配装一览
    • Meta Builds
    • Offmeta Builds
    • Other Builds

配装一览

碎层配装 通常分为三种角色定位：输出（DPS），进攻型辅助 和 防御型辅助。Meta 队伍只由输出和进攻型辅助构成。

  • Meta 是在队伍中表现最佳的 Builds，使用 Meta 的目标是最快最效率地完成所有碎层日常。相应地，难度也相对较高。
  • Offmeta 是用于替代 Meta 的 Builds，仍然具有出色的表现，但在理想情况下的表现弱于 Meta。
  • Other 是用于 T4 难度使用的 Builds，不推荐在挑战模式中使用。

Meta Builds

DPS 辅助
直伤魂兽师
1. 上手困难，也难于精通；
2. 高额伤害；
3. 可以延长自身和队伍的增益持续时间；
4.需要对战斗流程有一定的理解来调整站位；
5. 在非固定队中使用会困难一些。		 直伤敏捷龙魂使
1. 上手容易，难于精通；
2. 在Meta 队伍中是伤害最低的，但是这部分伤害仍然很重要；
3. 为队伍提供 敏捷，威能 和 保护；
4. 为队伍带来强大的 Buff： 和 。
直伤编织者
1. 上手难度中等，难于精通；
2. 高额、持续的伤害，非常多的闪避；
3. 依赖其他职业提供增益，如：敏捷，急速；
4. 通过炸火圈提供威能 和 激怒（甚至在战斗中）；
5. 几乎在所有 Boss 战中位居输出榜首。		 直伤插旗狂战士
1. 上手容易，难于精通；
2. 如果操作得当，在许多战斗中输出接近 直伤魂兽师 和 直伤编织者；
3. 为队伍带来旗帜 Buff： 和 。
直伤急速燃火者
1. 上手难度中等；
2. 在辅助中造成第二高的伤害，高于直伤敏捷龙魂使；
3. 负责提供 急速，圣盾 和 威能；
4. 在必要时可以提供 稳固 和 抗性；
5. 可以通过使用 提供 CC 和队伍增伤；
6. 在高水平的速刷队伍中，你可以切换成 直伤猎龙者 来应对 99CM；
7. 在极高水平的队伍中，由于可以维持 圣盾，你甚至可以使用 守护者
来造成更高的输出。

Offmeta Builds

DPS 辅助
直伤全息师
1. 上手困难，且难于精通；
2. 高额、持续的伤害；
3. CC 在 Offmeta Dps 中是最高的。		 症状急速燃火者
1. 容易上手；
2. 强大的爆发伤害，在非 Meta Builds 的辅助中拥有最高的伤害；
3. 负责提供 急速，圣盾 和 威能；
4. 在必要时可以提供 稳固 和 抗性。
直伤夺魂者
1. 上手容易，难于精通；
2. 高而持续的伤害，优秀的清怪能力；
3. 能够为自身提供增益，例如 威能 和 急速；
4. 在非 Meta Builds 的 DPS 中具有最强的生存能力；
5. 拥有 剥夺增益（Boon Strip） 的能力来应对 雾锁异变
福祸难料 （通常称为 No Pain, No Gain 或者 NPNG )。		 治疗敏捷龙魂使
1. 中等的学习难度；
2. 低伤害，高治疗；
3. 负责提供 提供 敏捷，威能 和 保护；
4. 可以调整特性，为队伍带来 ；
直伤猎龙者
1. 上手容易，难于精通；
2. 在短阶段战斗中造成中等水平的伤害，
在战斗流程较长时将会落后于其他 DPS ；
3. 利用 格挡 和 圣盾，具有非常高的生存能力。		 治疗急速燃火者
1. 容易上手；
2. 几乎零伤害，高而持续的治疗；
3. 负责提供 急速，圣盾，再生 和 威能；
4. 在必要时可以提供 稳固 和 抗性。
5. 可以通过使用 提供 CC 和队伍增伤。

Other Builds

DPS 辅助
直伤暴风使
1. 中等的学习难度；
2. 极其优异的清怪能力，成群小怪的克星；
3. 能够为队伍提供增益，例如 威能 和 激怒；		 直伤插旗战士
1. 容易上手；
2. 伤害比 直伤插旗狂战士 要低得多；
3. 为队伍维持 25 层 威能；
以及旗帜 Buff： 和 。
直伤独行侠
1. 容易上手；
2. 持续伤害能力，非常多的闪避；
3. 能够提供 隐身 来让队伍跳怪；
4. 有非常多的跳关能力，但是需要学习。
