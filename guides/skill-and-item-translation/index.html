GW2 Zh-CN 激战2 物品/技能翻译插件

2020-11-22

这个插件可以把 Metabattle, Snowcrows, Discretize 等激战2攻略网站上面的英文物品/技能描述替换成中文，方便美服/欧服玩家参考。

首先，打开你的电脑，台式也好，笔记本也罢，总之要有台电脑。

  • 1. 下载油猴扩展程序 Download Tampermonkey
  • 2. 下载莉莉编写的油猴脚本 Download Script
  • 3. 收工 End

1. 下载油猴扩展程序 Download Tampermonkey

如果你已经安装了下面列表中其中一个浏览器，请点进浏览器对应的页面，在「下载」选项卡选择 左边 的那个版本，安装。

  • 火狐浏览器 Tampermonkey on Firefox
  • 谷歌浏览器 Tampermonkey on Google Chrome（打开谷歌浏览器 Chrome 商店需要科学上网）
  • 微软 Edge 浏览器 Tampermonkey on Microsoft Edge
  • 苹果电脑 macOS Tampermonkey on Safari

如果你搞不懂上面在说什么，请按最简单的方法来操作，下载 Firefox 浏览器，然后在 Firefox 浏览器中打开本网页 Tampermonkey on Firefox。

2. 下载莉莉编写的油猴脚本 Download Script

用你刚刚已经安装过 Tampermonkey 的浏览器打开 https://greasyfork.org/zh-CN/scripts/377052，点击大大的绿色按钮「安装此脚本」。

3. 收工 End

打开对应页面，看看物品/技能有没有被翻译。如 https://metabattle.com/wiki/Build:Mirage_-_Greatsword_Farmer
