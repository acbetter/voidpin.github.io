跑图探索插件TacO

Tactical Overlay （也称为 TacO）是激战 2 的安全合法插件，是叠加在激战 2 上面的一个透明程序，与游戏界面完美集成。TacO 可以在游戏屏幕上提供标记对游戏进行引导（图标和说明文字）。

  • Taco能干什么？

    • 跑图路线引导（萌新好帮手，跟着路线走）；
    • 跳跳乐路径引导；
    • 各类采集点标记，包括各地图代币的采集；
    • 各个收藏成就的物品标记和成就指引（如白银荒地硬币）；
    • 剧情成就物品提示；
    • 公会任务指引；
    • 等等。

使用说明

  • 安装

    下载解压所有文件，单独放在任意文件夹中，推荐纯英文路径。

  • 添加扩展包

    1.下载上方链接中的扩展包，一般为 ZIP 文件；
    2.直接将下载完成的 ZIP 文件放到插件文件夹中的 "POIs" 文件夹内即可；

  • 使用

    1.先打开激战 2;
    2.把游戏设置窗口全屏或者窗口模式（F11）；
    3.打开文件夹中 "GW2TacO.exe" 启动插件；
    4.打开后在 PVP 图标附近有个饺子图标，点击后展开界面，选择相应功能，调整要显示的标记。

  • 卸载

    1.关闭插件;
    2.直接删除插件文件夹。

DPS监控插件-Arcdps

用于监控自己和队伍成员的Dps。

下载和使用

  • 下载地址

    • 下载名为 d3d9.dll 和 d3d9.dll.md5sum 的两个文件，并放入游戏文件夹 \Guild Wars 2\bin64 内。
    • 在游戏中使用 SHIFT+ALT+T 呼出面板。

Yolomouse鼠标指针

YoloMouse 是一款替换鼠标指针图标的插件，让你的鼠标变得非常醒目，避免激烈战斗中丢失目标。

下载&使用

  • 下载地址 点击橙色的 Download 按钮下载。

    1. 下载、安装并运行（ Win10 操作系统需要管理员权限）；
    2. 启动游戏；
    3. 选择要替换的鼠标指针，按 CTRL+ALT+1 选择偏好的指针样式。
