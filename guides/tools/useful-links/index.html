英文网站

数据库

  • 官方维基 : 最全最正式的官方数据库，你想知道关于激战 2 的一切；你也可以在游戏聊天窗中输入 /wiki 直接跳转。

查看Meta事件、世界Boss时间

  1. 维基事件时间表；你也可以在游戏聊天窗中输入 /wiki et 直接跳转。

  2. GW2 Timer：除了查看时间表，还可以查看个人账号内的所有资料，并支持网页版中文界面；

职业配装 Builds

  1. Metabattle：综合性的 Build 网站，包含 野外（Open World）、迷雾碎层和地下城（Fractal &Dungeon）、团队副本（Raid）、战场（WVW） 和 竞技（Conquest &PvP） 全方位覆盖的各类职业配装 Builds，但是在专业性方面稍逊于下面几个网站；
  2. Discretize(DT)：欧服顶级碎层公会 DT 创建，包含顶尖 PVE 玩家测试优化完毕的迷雾碎层 Builds，各个碎层打法以及一切碎层玩家应该了解的内容；
  3. Snow Crows(SC)：继 [QT] 之后，当前欧服顶级 Raid 公会 SnowCrows 的官方网站，包含最优的 Raid Builds，各个职业 Build 的专属攻略打法等；
  4. GodsofPvP：欧服顶尖PVP 队伍 参与制作的 PVP Builds，激战2 PVP 职业配置和打法的引领者。

成就攻略

  1. Metabattle：Metabattle 网站上还有各类成就攻略参考，按照时间顺序更新，相当便利，是目前搜索成就攻略的最佳选择。
  2. Dulfy：虽然由于种种原因，Dulfy停止了她个人攻略网站上关于激战 2 内容的更新，但是作为曾经最全最强大的激战2 成就攻略网站，它在 2019 年 5 月之前的大量高价值成就攻略仍然可以使用。

功能性网站

  • Efficiency：综合性的个人工具网站，提供查看账号内仓库、材料、角色各类物品价值，搜索道具位置，计算制造所需成本、罗列材料清单等等，功能丰富而强大。

    需要绑定账号 API：

    1. 游戏内按 ESC 点击 Edit Account 登录官网;
    2. 选择Applications-New Key-全点钩，生成 API 并复制；
    3. 登陆 Efficiency 网站，将你的个人 API 绑定到你的Efficiency账号上；
    4. 好了，你已经可以使用 Efficiency 的各种强大功能了。

  • FarmingCommunity：欧服最大的 Farm 社区，包括刷图收益排行榜、Meta/世界 Boss 收益、各类采集点收益、拆解收益计算和各类货币转换收益等等，一切以数据说话，是你刷材料赚钱的良好参考。

  • GW2 Timer：除了查看时间表，还可以查看个人账号内的所有资料，并支持网页版中文界面。

副职业升级

  • GW2Crafts：最实用的副职业升级指导网站。

GW2 Timer 使用攻略

由于激战 2 的数据库实际上存在各个语种的数据，而GW2 Timer网站可以借助这一点实现在线查看中文界面和各类物品的功能。

查看中文

  1. 打开 GW2 Timer；
  2. 在右上角的时钟附近，你可以找到语言选项，选择“简体中文”；
  3. 现在界面已经变为简体中文了，具有查看地图和世界boss时间等功能。

查看你账号

  1. 生成并复制你的 API （请参考上文 关于API的部分)；
  2. 在GW2Timer上绑定你的 API:
    • 点击“账户”，找到“API 索引键”；
    • 粘贴你的 API，接着在后面的栏里输入你想要命名的名称，然后打勾；
  3. 大功告成！你可以查看自己账号下各个角色、背包、银行和成就等一切了。

中文网站

中文数据库

  • 星岬岛 : 迄今为止国内最强大的数据库，数据齐全、中英文对照；

中文事件时间表

  • 星岬岛：使用了 Gw2Timer 网站的内核实现的中文事件时间表，在语言上更加便利；

中文成就攻略

  • 明眸：可供参考的中文攻略站；中文攻略相比英文攻略更加便利。

中文其他

  1. NGA：资料和攻略涵盖范围相当广泛，相当具有参考价值；但是也存在部分时效性不强的攻略，请自行甄别并使用。
  2. 金锟视频攻略：多种跑图、成就攻略，很适合初学者使用。
