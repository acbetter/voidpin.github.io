1. 什么是供给官徽记

PVE 传奇护甲： 完美特使护甲 的每一个部件都需要 1 份 ；

每 1 份 的制作需要 1 个 （50 个 兑换）。

在特定的 NPC 商人处用某些武器或者物品兑换。

参考：https://wiki.guildwars2.com/wiki/Faction_Provisioner

2. 这篇指南会告诉你什么

如何尽可能省钱地来获取 。

3. 计算依据

我们首先要参考 维基链接 了解兑换 所需的物品；

然后利用 gw2efficiency.com 计算成本的功能，添加所有列表中的物品，得到 各武器价目表 来计算最优解。

3.1. 使用方法

  • 各武器价目表 使用方法：

    • 打开链接后，找到 “Recipe tree” 字样的位置，点击下方的 Craft all 按钮；
    • 然后比较 TP 和 CRAFTING 价格，每种同名前缀的武器中，选择最便宜的那一把；
    • 有些特殊前缀的武器成本很高，我们就不做，直接跳过。
    • 一般来说，一把武器的制作成本在 20-40 之间都可以接受，超过 50 的就不划算了。

3.2. 其他物品的兑换

  • 考虑到 3x 可以合成 1x （可以使用 250 次）；后者与 等效。

  • 而 使用 250次的成本为： 60/次 x 250 次 = 1.5；

因此，每个 近似等值为 ，比起武器还是贵了点；

当然，如果你想加快供给官徽记的获取，使用 兑换也未尝不可。

4. 结论

综合上述，建议你按照如下方式兑换供给官徽记：

  • “3.2” 部分中的物品每天必换；
  • “3.1” 部分中的武器选择成本合适的，为了方便起见，应批量制作，每天兑换。
