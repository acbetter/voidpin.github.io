1. Crafting 制作简介

Crafting 工艺/制作（副职业） 可以允许你将各类材料制作成物品。副职业共有 9 种制作。

本篇内容主要搬运官方 Wiki，并添加了一些个人理解。完整内容请查阅官方维基 Wiki - Crafting。

如果你打算在欧服/美服冲制作，请不要看国服攻略，请不要看国服攻略，请不要看国服攻略，重要的事情说三遍！！！

2. Crafting experience modifiers 制作经验加成

在你冲某一项制作的时候，请把 制作经验加成拉满，具体可以看 P1 金昆 激战2 制作前必须 准备事项，但是视频的内容为国服制作，和美服不太符合，所以请参考下表把加成拉满！

效果 来源 加成 说人话
Item Booster Item Booster 50% 送的，没有的话可以拿 1桂冠换
Guild Crafting Boost Guild Crafting Boost 6-20% 视当前公会等级，需要去公会大厅吃
Productive Downtime Productive Downtime 10% 见 专精一览 中的 连续生产 专精

3. Level Up 怎样冲制作 1-400 及 400-500

todo：制作界面图

在欧/美服冲制作最好参考 ACCGs for Guild Wars 2 - http://gw2crafts.net，一位大佬写的脚本，计算出当前时间最省钱的冲制作配方及所需材料，然后你最好买了资料片并且过了七天（解封拍卖行所有权限，不然有些东西拍卖行买不了），背包里要有足够的钱（萌新肯定材料不够的，而且会缺很多很多），具体花多少 金币 可以参考下表。

学科 1-400 400-500
Armorsmith 35-45 15-25
Artificer 20-25 60-70
Chef 4-6
Huntsman 35-45 65-75
Jeweler 10-15
Leatherworker 35-45 45-55
Scribe 110-130
Tailor 35-45 20-30
Weaponsmith 40-50 60-70

然后下面这是这个网站主界面，你可以在 TOTALS 菜单查询当前时间所有职业冲级需要的金钱。提一句，不管是萌新还是大佬，都推荐使用 FAST GUIDES，NORMAL GUIDES 虽然看上去省钱一点，但是要浪费很多时间买乱七八糟的东西，把背包搞乱七八糟的，既浪费时间，又麻烦的很。当然，如果你想省钱的话，选择 NORMAL GUIDES 也是没有问题的，不过最好准备一个大点的背包来买各种材料，当然也得多准备时间。同理，冲烹饪也选 NO HEARTS，贵不了几个钱，方便快捷。

然后选择自己要冲级的制作，开始冲级！1-400 就选 FAST GUIDES，400-500 就选 400-500 对应的制作。

  • [0] 由于网页显示按照当前时间最省钱的冲级路线，一旦你刷新网页，网页的配方就可能发生变化，所以推荐大家不要刷新网页。举个例子，你一开始买好了材料 100 个布料，准备冲级，然后手贱点了一下刷新网页，新的网页需要 60 个布料和 30 个其他的东西，你不仅浪费了 40 个布料，还要去买一些其他东西，既浪费时间也浪费材料。
  • [2] 仅第一次填入 API 时有效，你在填入 API 之后再买的材料是不会显示的，所以请自己心里记好数量。
  • [3] 仅第一次填入 API 时会改变。萌新不懂 API 的请直接忽略。
  • [4] 红色数字代表需要购买的数量，黑色数字代表你当前拥有的数量（如果你填入 API 的话）。
  • [5] 供货商人即为制作师傅，一般在制造台旁边，制造台也一般在师傅旁边。如果你连这些都找不到的话，还是老老实实看金昆视频攻略吧，P2 烹饪、P3 珠宝、P4 工艺、P5 裁缝、P6 猎手、P7 皮革、P8 武器 和 P9 盔甲（抄写员在高级公会里做，萌新基本用不到）。

  • [9] 一般情况下，Tier 1 阶段会消耗大量 Tier 1 各制作对应材料。

  • [11] 发现配方去要去用鼠标点击工作台界面左边侧栏 Discover 的选项卡，然后把材料放入即可发现配方。
