请在游戏内按 ESC 打开设置面板，一边阅读本文一边按照自己的需要进行设置。

基础

常规设置 General Options

用户界面 User Interface

  • 显示所有敌人名字 Show All Enemy Names: 选中 ✔
  • 显示技能冷却 Show Skill Recharge: 选中 ✔
  • 小队生命条加粗 Thick Party Health Bars + 总是显示小队生命条 Always Show Party Health Bars: 选中 ✔ - 可以更轻松地查看队友状况，在玩辅助/治疗的时候特别有用；
  • 显示简单症状提示 Show Simple Condition Floater: 选中 ✔ - 减少视觉干扰。

视角 Camera

  • 旋转速度 Rotation Speed: 最大（拖动到最右） - 默认速度实在是太慢了。
  • 视野 Field of View: 最大（拖动到最右） - 可以让你自由地调整视角。
  • 关闭玩家视角抖动 Disable Player Camera Shake: 选中 ✔
  • 使用自由视角 Use Free Camera: 选中 ✔

战斗/移动 Combat and Movement

  • 技能定点指向 Ground Targeting: 带有距离指示器的快捷施法 Fast with range indicator - 只要按住技能键就可以选择技能施放位置，并在松开技能键时立即施法。如果不需要额外瞄准并且希望技能尽快施放，请立即松开技能键。高效又便捷。

  • 双击方向键闪避 Double-Tap to Evade: 不勾选 - 没有人希望莫名其妙跳崖或者浪费耐力条翻滚，请为翻滚设置一个键位。

  • 禁用范围效果提示 Disable Area of Effect Rings: 不勾选

  • 自动选择目标 Autotargeting: 可选 - 但这个功能往往会影响一些移动技能，导致你在想要逃跑时反而冲向敌人，例如 。你可以 不勾选。

  • 技能目标提示 Promote Skill Target: 不勾选

  • 近战攻击辅助 Melee Attack Assist: 不勾选 - 防止你离开近战攻击范围，但实际上笨重又影响操作

  • 将定点指向技能锁定到最大距离 Lock Ground Target At Maximum Range: 选中 ✔ - 如果你要使用 这类技能移动，使用此选项允许你可以不必担心点击的位置，在直接施放该技能时获得最大移动距离。

  • 使定点指向技能对准当前目标 Snap Ground Target To Current Target: 不勾选 - 只会干扰你的技能施放。

  • 允许技能改变目标 Allow Skill Retargeting: 选中 ✔ - 在一般情况下可能不重要，但是还是有必要说明这个功能，因为它在 PVP 里非常有用。它允许你改变技能目标，在技能引导过程中为技能重新选择目标。比如向目标 A 使用技能，这可能会骗出翻滚，然后在最后一刻改变为目标 B（实际目标），令敌人措手不及。

竞技 Competitive

  • 标准敌人模型 Standard Enemy Models: 选中 ✔ - 你不想满地找模型超小的 阿苏拉 玩家吧？那就勾选上。
  • 禁用 PVP 队友生命条 Disable PvP Teammate Health bars: 不勾选
  • WVW 简单姓名板 WvW Simple Nameplates: 仅敌人 Enemies only
  • 禁用 WVW 领土浮窗 Disable WvW Territory Overlay: 不勾选

操作设置&键位 Control Options &Keybinds

移动 Movement

  • 向后转 About face - 对于使用需要快速旋转 180 度才能“向前”移动的技能时非常有用，例如幻术的 和神枪手的 。除此之外，它在PvP中还有特殊用途，例如你假装要从悬崖上往下跳，引诱敌人真的跳下来，而随后你在空中使用 About Face 回到了悬崖上。

技能 Skills

  • 总体原则：
    一般来说，尽可能将常用的功能性技能和动作按键放在左手区域，可能的话利用鼠标侧键。

  • 例如将 QERTG, ZXCVB 这些键都放上常用的技能，能为你的操作带来便利。

  • 请不要忘记设置 特殊行动键 Special Action。如果有鼠标侧键，推荐将它放在鼠标侧键上，这能允许你在移动鼠标选定技能落点时同时施法。

选择目标 Targeting

  • 标记目标 Call Target (CTRL+T) 和选择标记目标 Take Target (T) 在游戏中相当有用，你应该对其进行设置或者采用默认键位。

画面-进阶设置 Advanced Settings

  • 预设 Settings Preset - 建议先调整为 Autodetect 自动检测，然后逐项调整下方的画面设置；
  • 角色模型限制 Character Model Limit 和角色模型质量 Character Model Quality - 建议调整为 低Low 或者 最低Lowest，能够显著提升野外大型时间和战场的帧数，减少卡顿；但是在迷雾碎层和 Raid 中应该适当调高，避免由于角色模型限制过低导致某些关键技能区域不显示。
  • 小地图 - 如果你愿意的话，可以把鼠标移动到小地图右侧栏变成手形，将小地图从默认位置（右下角）拖动到游戏界面的右上角。

其他 Miscellaneous

  • 收起/拔出武器 Stow/draw weapon - 在 PVP 中非常重要。由于游戏内没有“取消施法”的键位，需要这个键位去实现取消施法的功能。它有以下优点：
  1. 保留技能 CD，防止你因为被 致盲 或者目标 脱离视野/超出范围/正在闪避/无敌/等 原因而导致技能无法命中的情形下浪费技能；
  2. 用于取消某些技能的后摇；
  3. 在 PVP 中骗取敌人的翻滚，比如你在施放 时会有明显的动画，假如敌人翻滚，你可以用 收起武器 键取消技能施放。

可选

上文中不包含的一些优先级稍低的选项。

常规设置 General Options 2

用户界面 User Interface 2

  • 显示时钟 In-game clock: 本地时间 Local time - 可以很好地和 事件时间表 结合使用，帮助你按照计划进行游戏。
  • 内容向导 Content Guide: 默认 Default - 帮助你追踪事件或者目标，你也可以自己调整
  • 忽视先前完成的目标 Ignore Previously Completed Objectives: 选中 ✔ - 这会阻止内容想到将你引向已经完成的爱心和技能点。
  • 互动时范围拾取 AoE loot on interact: 选中 ✔
  • 自动拾取 - 自动 Autoloot - Autopickup: 选中 ✔
  • 自动拾取 - 快速互动 Autoloot - Quick Interact: 选中 ✔
  • 显示所有敌人名字 Show All Enemy Names: 选中 ✔
  • 显示所有 NPC 名字 Show All NPC Names: 选中 ✔
  • 显示所有玩家名字 Show All Player Names: 选中 ✔
  • 显示所有可用物体 Show All Usable Object Names: 选中 ✔
  • 总是显示我的名字 Always Show My Name: 可选
  • 禁用 ESC 窗口关闭 Disable Closing Windows with ESC: 不勾选
  • 显示目标生命百分比 Show Target Health Percent: 选中 ✔
  • 移动整理物品选项 Move Inventory Compact Option: 可选 - 如果你想移动“ 存放材料 Deposit Materials” 这个选项的位置的话。
  • 自动展开搜寻队伍预览 Automatically Expand LFG Preview: 选中 ✔
  • 隐藏可选公告 Hide Optional Announcements: 不勾选
  • 团队生命条加粗 Thick Squad Health Bars: 不勾选
  • 总是显示团队生命条 Always Show Squad Health Bars: 选中 ✔

视角 Camera 2

  • 水平位置 Horizontal Position: 调整到正中
  • 纵向位置（近） Vertical Position Near: 最小（拖动到最左）
  • 纵向位置（远） Vertical Position Far: 调整到正中
  • 碰撞灵敏度 Collision Sensitivity: 可选 - 你可以调整并选择适合自己的位置
  • 缩放灵敏度 Zoom Sensitivity: 最大（拖动到最右）
  • 反转视角 Y轴 Invert Camera Y-Axis: 不勾选
  • 开启视角抖动 Enable Camera Teleportation: 不勾选 - 很容易晕，关了吧
  • 使用任一鼠标键改变方向 Use either mouse button to change direction: 不勾选
  • 启用第一人称视角 Enable First-Person Camera: 选中 ✔ - 如果你想截图，这个选项可以一直缩放。
  • 将视角调整为角色高度 Adjust Camera to Character Height: 不勾选

战斗/移动 Combat and Movement 2

  • 双击进行攻击/互动 Double-Click to Attack/Interact: 可选 - 一般来说不会用

  • 点击右键进行攻击或互动 Right-Click to Attack/Interact: 选中 ✔ - 如果你同时处于倒地的队友和倒地的敌人中间，游戏会自动提供 “终结敌人 Finish！”或者 “复活队友 Rez ”选项（靠近的优先），这时候你可以用右键选定目标，避免误操作。

  • 改变目标时停止自动攻击 Stop Auto-Attacking on Target Change: 不勾选

竞技 Competitive 2

  • 队伍颜色 Team Colors: 选中 ✔
  • 标准敌人模型 Standard Friendly Models: 可选 - 如果你尽可能地排除视觉干扰，或者只是讨厌过多的光环/传奇效果，那么一定要选中此选项。

同步下载 Streaming

  • 在游戏中同步下载数据 Content Streaming: 可选 - 根据你的网速决定。

小组 Groups

  • 显示所有指挥官标志 Show All Commander Tags: 选中 ✔ ；在 WVW 中是 可选 的，选择关闭可以更轻松地查看队伍或追踪队友的移动。
  • 简化队伍界面 Simple Party UI: 选中 ✔ - 并不是必须的，如果你希望 PVP/WVW 的面板占据更少的空间，可以勾选。
  • 清除全部团队标记 Clear All Squad Markers: 可选 - 适用于经验丰富的指挥官。

坐骑 Mounts

  • 禁用条件满足时坐骑移动能力输入 Disable Conditional Mount Movement Ability Input: 选中 ✔ - 防止坐骑自动使用其移动能力，而是使用跳跃键手动使用。 坐骑：允许视角滚动 Mount: Enable Camera Roll: 可选 - 可能导致晕眩。

操作设置&键位 Control Options &Keybinds 2

选择目标 Targeting 2

  • 最近的敌人 Nearest Enemy: 请为该功能设置一个键位
  • 收起武器 Stow Weapon: 请为该功能设置一个键位

团队 Squad

  • 新手可以跳过此部分，但任何想参与队伍指挥的人都应绑定团队标记快捷键。
