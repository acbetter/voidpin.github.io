每日日常的奖励

每日日常是游戏内的高收益内容之一，分为 PVE，PVP 和 WVW 三类，每天刷新。 如何最快速地完成日常获得每日收益？首先先了解一下每日日常的奖励：

类别 主要奖励
PVE ;
PVP ;
WVW ;

在完成任意三个日常后，你将额外完成 日常完成者 Daily Completionist 成就。

  • 奖励的大宝箱中含 2 金币 和 3 灵魂碎片 ，这是重要的 金币 来源之一。

为什么我要做 PVP 或/和 WVW 日常并积攒奖励药剂

  • 使用1瓶奖励药剂可以为奖励分支增加 1.25% 的进度（80 瓶药剂= 100% 进度），积攒奖励药剂可以将 PVP / WVW 的奖励分支“储存”起来；当需要时，你可以通过消耗积攒的奖励药剂快速获取某一个奖励分支的奖励。

奖励分支如何选择

  • 如何选择分支？下表以 WVW 奖励分支为例， PVP 奖励分支同理。

完整的奖励分支说明请查阅官方维基 https://wiki.guildwars2.com/wiki/Reward_track

各分支收益一览请查阅： https://fast.farming-community.eu/wvw/reward-track

分支类别 说明
不可重复的奖励分支 仅可完成一次。
完成时一般能获得大量 和价值很高的材料。
动态世界第四季奖励分支 (Living World Season 4 tracks) 赚钱。每整条分支的材料收益大约 25 金，
最终奖励里的装备卖给 NPC 可以获得 2.5 金。
万圣节/冬幕节节日奖励分支 赚钱。
地下城奖励分支 (Dungeon tracks) 完成收藏成就，获取成就点数。获得的 橙装 可以拆解。
迈古玛丛林/迈古玛荒地奖励分支 获取地图代币。
水晶沙漠奖励分支 完成收藏成就，获取成就点数。
战斗礼赠奖励分支 获取制作传奇需要的 。

有哪些容易完成的日常

完整的日常成就攻略请查阅官方维基 https://wiki.guildwars2.com/wiki/Daily_(achievements)。

PVE

名称 难度 说明
Daily Gatherer 简单 在日常提示的地区采集植物、木头或矿石，
家园副本和公会大厅采集也计入相应地区的成就。
Daily Activity Participation 简单 参加一个 活动。
（在狮子拱门等地图里找到 图标的 NPC 参加）
Daily Mystic Forger 必做 使用一次神秘熔炉。
最简单的就是放四件蓝装进去合成。
奖励 1 个 。
Daily Vista Viewer 简单 去日常提示的地区观看一个风景点。
Daily Jumping Puzzle
Daily Mini-dungeon		 简单/中等 你可以去成就里指示的地图碰碰运气，
经常会有人开传送门，这样的话日常会很容易完成。
完成跳跳乐/迷你地下城还可以拿成就点数。

PVP

名称 难度 说明
Daily PvP Matches Played
in Unranked or Ranked Arena		 简单 完成任意一场 PvP 比赛，非排位和排位均可。（打就行了）
Daily PvP Rank Points 简单 在PvP比赛中获得50 Rank点数（PVP等级的经验值）。（打就行了）
Daily PvP Reward Earner 简单 通过PvP奖励分支获得1个奖励。（打就行了）
Daily PvP Player Kills 中等 在PvP比赛中击败或帮助他人击败3名敌方玩家。

PVP内经常会有互刷房，可以很容易地完成日常。如下图中房间名带有 “Rankfarm” 或者 “Farm” 的都是。有的互刷房不允许杀人或者需要提前询问，请注意房间规则。

WVW

名称 难度 说明
Daily WvW Big Spender 简单 花费 25 个 购买物品 说明 。
Daily WvW Caravan Disruptor 简单 截杀一头 牦牛。
Daily WvW Land Claimer 简单 击杀哨兵并占据 哨点。
Daily WvW Master of Monuments 简单 在边境战场占领一处遗迹。
Daily Mists Guard Killer 简单 击杀 5 个NPC守卫。
（可以在占领营地的同时完成）
Daily WvW Veteran Creature Slayer 简单 击杀 精锐座狼，精锐鹰身女妖 或者 精锐地虫。
Daily WvW Camp Capturer 中等 占领 2 处 营地。
  • 说明： 完成该日常最简单的方式是在战场商人处购买 攻城蓝图 或者在公会大厅里购买 。
    你也可以在 WVW 桂冠商人 处购买 灌注 ；如果你不清楚购买哪个，购买 总是没错的。
