服务器选择与转服

1. Start 101

对于 PvE/PvP 玩家，服务器只有两个划分，一个是美服，另一个是欧服。各区域内的小服务器可以相互跨服组队玩耍。

对于 WvW 战场玩家，各区域内的小服务器则相互敌对。请谨慎选择服务器，不同服务器的活跃时段、强弱各不相同。

2. Intro 服务器机制介绍

2.1. Home World 家园世界

你在 WVW 中将代表服务器而战。

玩家在第一次登录游戏时选择一个 服务器 作为“家园世界”；这决定了你在 世界之战（WVW） 中所代表的服务器。你所有的角色都将属于该家园世界。你可以通过支付 宝石 来转服。

2.2. Region 地区

指代玩家游戏数据的存放位置，分为北美地区（美服）、欧洲地区（欧服）和国服，玩家无法进行跨区域游戏，也无法与不同区域内的玩家进行互动。

只有欧服和美服之间的拍卖行是互通的，并且可以互相转服。

2.3. Megaserver 超级服务器

PVE 和 PVP 不受服务器的限制。

与其他网络游戏不同，激战2采用镜像分流和借服假排机制，同地区的所有玩家都可在超级服务器的作用下自动进入同一地图中进行游戏与互动。当某一地图容纳玩家人数达到上限时，超级服务器会不断建立新的镜像分流来接纳更多的玩家，不同的镜像分流内地图机制完全相同，但处于不同分流内的玩家之间无法共同游戏。通常情况下，服务器会优先将同队伍、同公会、同服务器、同语种的玩家放入相同的地图分流。当同队伍玩家处于不同地图分流时，在队伍UI中你将无法看到队友的头像，也无法与队友共同游戏，这时可以通过鼠标右击队友名片打开队伍UI界面，选择“加入[地图分流名称]”以进入到队友所在分流，但当其所在分流人数已达到上限时则无法加入成功。

3. Server Selection 服务器如何选择

选择欧洲地区并按需求选择服务器。

首先在登陆后的人物选择界面，你在左下角可以看到：

此时，点击 “World Selection（世界选择）”按钮打开面板，在右上角的 “Location（地区）”里选择“Europe 欧洲”；

其中，服务器名字右边有 [DE][FR][SP] 后缀的分别代表德语、法语和西班牙语服务器。如果你不是这些语种的使用者，建议你不要选择这些服务器。

在右侧，你可以看到服务器的的人口状况： Full 已满； Very High 非常高； High 高； Medium 中。

目前我们常驻服务器为 Whiteside Ridge [WSR]。

4. Transfer 转服

参考：https://wiki.guildwars2.com/wiki/World#Transferring

要转移到另一个服务器，请在世界选择界面选择相应的服务器然后点击右下角的“Transfer （转服）”；你将在转服过程中暂时离线，转服可能需要 5-30 分钟。你可以每七天转服一次，费用由目标服务器人口状态决定。免费帐户不能转服。

在转服后你将暂时无法获得 WvW 分数，这持续到两次 WvW 重置之后（至少 7 天，最多不超过 14 天）。没有其他惩罚。

转服 需要费用，根据人口负载而定：

  • Medium 中：500
  • High 高：1000
  • Very High 非常高：1800

4.1. Free Transferring 免费转服

另外，如果你将所有人物删除，你可以进行一次免费转服，该操作每七天仅限一次且需要目标服务器负载情况允许。你的公会、货币、银行和材料仓库将会保留。

如果你这样做，你所删除的角色无法被全部或部分找回。
