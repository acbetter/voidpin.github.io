1. 简介

维基链接：https://wiki.guildwars2.com/wiki/Strike_Mission

进攻任务 是 1-10 名玩家的团队可以参与的 PVE 副本，难度介于开放世界（Open World）和团队副本（Raid）难度之间。每个任务只有一场 Boss 战。

所有进攻任务都可以从 北方之眼（Eye of the North） 进入。请熟练使用 LFG 寻找并加入队伍。

如果不熟悉某些术语，请查阅 常用游戏术语。

1: 仅在冬幕节可玩。

每个 Boss 的战斗都会在右侧有一个倒计时，越快击杀 Boss，你能获得的奖励就越丰厚。

2. 奖励

你需要坚韧精华操控2级、勇气精华操控2级和警惕精华操控2级才能开启某些宝箱。请查阅 专精一览。

2.1. 先知水晶

每次进攻任务都会奖励你先知水晶碎片 //。

这些碎片可以用来从 晶体学家 Smoxxi 那里购买升华装备，也可以用来交换完整的先知水晶。

先知水晶可以用来打开 使者宝箱：一个每周可以打开一次的大宝箱，位于北方之眼传送门附近。

2.2. 每日优先进攻任务

完成每日优先进攻任务会奖励你一个宝箱，里面必定包含一个先知水晶。每周你可以以这种方式最多获取 3 个先知水晶。

3. 推荐配装

与团队副本类似，进攻任务需要一个有组织的团队才能顺利通关。虽然要求并不那么严格，但是总体思路是一致的。请查阅 团队副本：入门。

所有 团队副本：入门配装 中的配装都可以在进攻任务中使用，你可以按照喜好来选择。

由于进攻任务的难度和门槛都较低，所以由 敏捷龙魂使 和 治疗急速燃火者 构成的 “ Firebrigade 消防队” 组合作为关键增益 急速 和 敏捷的提供者，能够大放异彩。

在治疗角色中，治疗德鲁伊、治疗急速燃火者、治疗风暴使 和 治疗灾厄师 都是非常好的选择。尤其是 治疗风暴使 能够使用 抵挡致命伤害； 治疗灾厄师 利用 特性挽救倒地的队友，在野队中很受欢迎。

至于输出角色，在达到一定水平后，推荐使用 症状急速燃火者 和 症状龙魂使，因为这两个职业的功能性和输出能力强大，并且适用于目前所有的进攻任务。

4. 攻略

4.1. 席瓦雪山小径

简易攻略：利用 稳固，合理翻滚和走位躲避各种 AoE 和砸地攻击，站桩输出。

与鲜血军团队长交谈，在附近开辟一条冰隧道。当沿着隧道前进时，团队将遇到迎面滚来的巨大雪球。被雪球击中会让玩家在回到隧道的入口并被连续击倒。但是在隧道的边缘有多个小凹陷，这些地方是安全点，可以用来躲避雪球。或者，在身上有 稳固 时，可以避免被击倒；建议同时利用翻滚来确保安全。

通过隧道后，有一片跳跳乐区域，其中有五个 戈瑟玛山谷宝箱 可以开启。

跳跳乐的某些地方非常滑，一旦摔倒你可能需要从头重新开始。当第一个玩家到达终点时，一个 传送灯塔 会被点亮，允许玩家在地图中点击并传送到 Boss 冰巢构造体 的平台上。

战斗一旦开始，玩家会持续受到 的伤害。

  1. 在战斗开始时，建议团队面向冰巢构造体在左侧抱团。这将使团队更容易地躲避 的冲击波，尽量减少无谓的移动。

    第三次冲击波攻击后，Boss 会释放暴风雪，吸附所有玩家围绕它旋转；在暴风雪暂时停歇时，团队可以移动到 Boss 右侧。

    低于 85%生命值后，Boss 会释放覆盖半个场地的 冰臂挥击，团队可以稍微向另一侧移动来躲避。

    低于 75%后，每 5 秒一个随机的玩家就会被一个中等范围的 AoE 攻击，该攻击会造成少量伤害，并在 3 秒内造成 冰冻。该机制持续到 50% 生命值为止。

  2. 当生命值降到 50%时，冰巢构造体的左臂将破碎，冰巢构造体将崩塌。随后，它会开始旋转，造成少量伤害并击退玩家。

    冰巢构造体会按照以下顺序施放技能： → 逆时针旋转 → → 逆时针旋转 → → 逆时针旋转 → 致命寒冰波动 → 重复。

    这些攻击的伤害很高，但不会立即让人一击就从满血到倒地。玩家应该合理使用翻滚，同时团队利用保护性增益来应对，例如 稳固、保护 和 屏障。

4.2. 亡者之音和亡者之爪

简易攻略：先攻击弓手，使用防御投射物技能抵挡它的箭矢，等锤手靠近后一同输出两个 Boss；Boss 传送后观察两者的血量，必要时转火。有蔑视时破蔑视，尽量将他们同时击败。

当第一个玩家接近其中一个 boss 时，战斗将开始。整场战斗被分成多个 30秒的小阶段。亡者之音和亡者之爪将战斗30秒，然后传送到新的位置，在那里他们将重新与玩家交战 30秒，如此循环。当第一个 boss 倒下时，第二个boss 会传送到竞技场的中心并狂暴，它的攻击会变得更加危险。因此，团队应该尽可能同时杀死两个 boss。

在整个战斗中，玩家每三秒都会受到来自 的少量伤害。竞技场的外部将被一个危险的区域覆盖，玩家进入其中会遭受每秒最大生命值的 20% 的伤害。

亡者之音 使用弓箭进行远程攻击，主要技能有 和 ，需要用 投射物防御 来应对。

亡者之爪 使用巨锤进行近战攻击，主要技能有 （造成击倒）和 锤击 （造成中等伤害），如果近战范围内没有目标，它会使用 飞锤 技能移动到玩家身边。

由于两个 Boss 一近一远的特点，团队会优先攻击弓手，等锤手移动到玩家身边后，同时攻击两个 Boss。

  1. 弓手在西，锤手在中。注意躲避锤手的 AoE 技能。

  2. 弓手在西，锤手在北。注意躲避锤手的大范围 AoE 攻击。

  3. 弓手在东，锤手在西。两个 Boss 使用组合技能“双重洪水”：亡者之爪将他的锤子抛向空中，亡者之音将一支箭射向它，引发从竞技场中心爆发的爆炸，对命中的玩家造成极大的伤害。合理使用翻滚和其他技能应对。

  4. 弓手和锤手在中间。两个 Boss 使用组合技能“致命协同”：亡者之音和亡者之爪联合他们的力量，获得一个 蔑视条 并制造成对的圆锥 AoE 攻击，留下伤害性区域。这些区域会持续 5秒，每隔 2.5 秒就会按照顺时针方向制造一对新的区域。这一技能会一直持续到 蔑视条 被打破，或者阶段结束。

  5. 弓手在西，锤手在东。该阶段没有组合技能施放，在结束后重新从第 1 步开始循环。

  6. 狂暴：一旦一个 Boss 死亡，另一个 Boss 将传送到竞技场中心，开始发动类似于 致命协同 的 AoE 攻击，但每次攻击范围都会覆盖竞技场的四分之一，并造成严重伤害，但可以通过顺时针旋转来避免。每第五次攻击，Boss 将释放一股类似于 双重洪水 的冲击波，并获得 5 层狂暴效果，造成的伤害进一步增加。

4.3. 卓玛爪牙法兰涅尔

简易攻略：带好 稳固 和格挡技能，合理利用翻滚或者跳起来躲避冲击波和各种 AoE ；帮助击碎冻住队友的冰块使队友挣脱；被小红圈点名分散；在中间输出 Boss。

战斗一旦开始，玩家会持续受到 的伤害。

  1. 开场时，法兰涅尔会高高跃起，使用 ，被冰柱击中的玩家会被冻在冰块里无法行动。这些冰柱的落点是固定的，注意观察。

    随后，法兰涅尔会化作乌鸦回到场地中央，施放 ；随后， 和 。合理地跳起躲避、翻滚，或者使用 稳固 来应对 击倒。

  2. 75%生命时，法兰涅尔会在施放一次 后消失，并召唤 冰巢构造体。

    它会使用 、冰臂挥击 和蓄力攻击 。通过 破蔑视 来打断最后一项技能（如果选择不破蔑视，冰巢构造体会跳过 70% 生命值时的无敌阶段）。

  3. 当达到 70% 的生命值时，冰巢构造体的左臂将粉碎，露出其中操纵着它的法兰涅尔。不久之后，它将崩碎，然后旋转并击退玩家。

    首先它会使用 ，召唤一群冰元素；随后使用“严寒轰击”技能：点名所有位于左侧的玩家，发射冰块投射物并造成伤害。

    如果玩家抱团在一起，会受到多个冰块的攻击倒地。最简单的应对方法是站到右侧；或者使用 投射物防御 来应对。

4.4. 骸骨剥皮怪

简易攻略：

使用合理的团队组合，3 个左右的治疗，症状输出较直伤有优势。全团尽量抱紧在一点，紧贴 boss 顺时针或者逆时针绕圈，躲避致命的大红圈和扇形吐息，同时输出 boss；小怪喷吐的小红圈可以硬吃。

Boss 起飞时，全团抱团覆盖保护性增益并刷血，抵挡它飞回来时砸地的伤害。特殊行动键破蔑视打断 Boss 技能。

在整个战斗中，玩家每三秒都会受到来自 的少量伤害。竞技场的外部将被一个危险的区域覆盖，玩家进入其中会遭受每秒最大生命值的 20% 的伤害。

  1. 战斗开始后，骸骨剥皮怪会根据已经熄灭的火把数量获得 的层数，造成全场脉冲伤害——一种应对方法就是让玩家去不断点燃火把。

    骸骨剥皮怪会持续施放“抓攫”技能：点名所有玩家并在他们脚下生成红圈，如果不及时躲避，玩家会立即倒地。、 等技能和特性可以有效地抢救弱智和手残的队友。

    骸骨剥皮怪还会每隔 15-20 秒施放一次 ：选定某一个火炬或者玩家（如果所有火炬已经熄灭），喷出一道狭窄的扇形吐息攻击，造成 击退。使用 稳固 或者 解除昏迷 来应对，否则你很可能会受到后续的红圈伤害而倒地。

  2. 从 90% 生命值开始，骸骨剥皮怪会每隔 30-40 秒获得 蔑视条，并在附近召唤 4-6 只小怪。使用特殊行动键破蔑视；小怪喷吐的小红圈可以使用 投射物防御 来应对。

  3. 在 75%、50% 和 25% 生命值时，骸骨剥皮怪会使用 ，高高跃起并拉拽所有玩家。所有火炬会熄灭，一个红圈会迅速扩散到竞技场边缘，此时，骸骨剥皮怪重重地落下砸地，对所有玩家造成高额伤害。

    骸骨剥皮怪还会重复使用 抓攫、、召唤小怪 和 残暴碾压（冲到场地边缘然后重重地跳回来砸地）技能。

4.5. 霜渊之音

简易攻略：

使用合理的团队组合，2 个左右的治疗，症状输出较直伤有优势。

橙圈分散，绿圈集合分担；

75% 前抱团贴boss输出，所有伤害硬吃；

75% 后Boss会一次点名 4 人生成 4 条锁链，中的人只允许前后移动，其他人躲开锁链，不然会暴毙。

最后 20% boss 会周期性全场放小球，接触到小球会生成一个爆炸区域，此时远程输出。

战斗一旦开始，玩家会持续受到 的伤害。

  1. 一旦与中间的冰块互动完毕（约 5 秒），所有玩家都会被击飞到空中（可以使用滑翔翼）；与卓玛低语重合站位的玩家会被击退但不会受到伤害。击飞不可被闪避。

    紧接着，卓玛低语会引导四个 ，造成中等伤害。随后，卓玛低语会使用 制造三个巨大的高伤害区域，以及 ，使用橙圈点名所有玩家，玩家应分散来躲避其他玩家的橙圈伤害，避免倒地。

    在这之后，卓玛低语会使用 并击飞所有玩家，并开始施放 ，玩家应抱团集合分担绿圈的伤害，利用保护性增益来应对，例如 稳固、保护 和 屏障；在特定时刻，被点名的单个玩家应该被牺牲。

  2. 75% 生命值时，卓玛低语会暂时消失，玩家暂时被冻结，无法进行任何行动。随后，玩家会被传送到一个小型竞技场迎战自己的复制体；所有玩家击败复制体后，卓玛低语重新出现。

    卓玛低语会使用两项新的技能：
    - ：一次点名 4 人生成 4 条锁链，锁链对其他玩家造成极高的伤害；因此中锁链的人只允许前后移动，不允许左右移动，其他人躲开锁链，不然会暴毙。
    - ：引导高伤害的全场 AoE，可以通过破蔑视来打断。

    在这一阶段，卓玛低语会依次采取以下行动：
    → → → 传送到边缘 → → 群体击飞 → → 传送到中央 → → 群体击飞 → 。

  3. 25% 生命值时，会再次出现一次复制体阶段。回到主场地后，卓玛低语将释放一道冲击波击飞玩家，但不会造成伤害；然后它开始快速旋转，连续释放 AoE，造成少量伤害。同时，Boss 的体型增大，并获得寒冰构成的武器。

    卓玛低语会使用一项新的技能：
    - “冰凌坠落 Slithering Rime” ：大量冰箭从 Boss 手中射向四面八方，随后，它开始快速旋转，向场地边缘发射大量球体；如果球体击中任何玩家或友方单位，它会在一个大范围内造成中等伤害。该攻击持续8秒。你可以使用 这类技能“引爆”球体。

    在这一阶段，卓玛低语会依次采取以下行动：
    → AoE技能1 → AoE技能2 → → → 冰凌坠落 或者
    → 群体击飞 → AoE技能2 → 冰凌坠落 或者 → 或者 。
