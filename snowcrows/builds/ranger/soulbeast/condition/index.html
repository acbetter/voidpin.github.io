Overview 概述

Condition Soulbeast 症状魂兽师 是一个以症状为基础的输出职业。该配装拥有简单的循环，很快可以上手，装备也相对容易获取。

尽管该配装的输出容易被其他循环更复杂的症状输出超过，但仍然是那些不想学习 Condition Renegade 症状龙魂使、Condition Firebrand 症状燃火者 或 Condition Mirage 症状幻象术士 的理想选择。

Condition Soulbeast 症状魂兽师 可以支援 Druid 德鲁伊 ，使用一个通用技能栏提供一个精魂而仅仅损失极少的输出，一般会在 、 或 中选择其一。
