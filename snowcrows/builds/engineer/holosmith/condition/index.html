Overview 概述

Condition Holosmith 症状全息师 是一个强大的以症状为基础的输出职业，提供一定的高爆发。

因为拥有高额的 Power 属性，该配装严重依赖暴击。由于大量的物理伤害，它同样会从低 Armor 首领战中获益，Vale Guardian 山谷守护者 是一个绝佳的例子。

Condition Holosmith 症状全息师 具有陡峭的学习曲线，不推荐初学者使用。

Notes 注释

你的循环大致如下：“Opening 开场” - “First Loop Rotation 第一次循环” - “Second Loop Rotation 第二次循环” - “Repeat from the First Loop Rotation 从第一次循环重复”。

  1. Opening 开场
  2. First Loop Rotation 重复循环一
  3. Second Loop Rotation 重复循环二
  4. Repeat from the First Loop Rotation 再次进行重复循环一

你可以在循环的第三步中使用 中断自动攻击链。你希望在尽可能接近 50 热量时这样做，这基本上是在 可以使用的同时。极偶然地，有可能出现你会没有完全过热的情况。如果你看到这种情况即将发生，只需要稍等片刻；但这种情况非常罕见。

只在施放时间结束时你恰好拥有 50 热量的情况下使用 。如果可以，延迟该技能的施放直到没有任何其他优先级更高的技能可以使用。
